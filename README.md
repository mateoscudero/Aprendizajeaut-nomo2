Proyecto Integrador – Juego del Ahorcado
Este proyecto consiste en el desarrollo de un juego programado en Python el cual permite aplicar los principios fundamentales de la lógica de programación mediante el uso de estructuras de datos, funciones, validaciones y control de flujo.
El proyecto demuestra cómo las nuevas tecnologías, específicamente la programación, pueden utilizarse para crear herramientas interactivas que apoyan el aprendizaje, el razonamiento lógico y la resolución de problemas, aspectos clave en el desarrollo tecnológico actual y futuro de la sociedad.

El objetivo principal del programa es:

Aplicar la lógica de programación en un proyecto funcional
Desarrollar un juego interactivo utilizando Python
Utilizar estructuras como diccionarios, tuplas y listas
Implementar funciones para organizar el código
Demostrar el uso de tecnología como herramienta educativa
Principales funcionalidades del código
Selección aleatoria de una categoría y palabra usando la librería estándar random

Validación de la entrada del usuario:

Solo permite letras
Evita letras repetidas
Control de intentos limitados
Visualización del progreso de la palabra
Mensajes de victoria o derrota
Opción para volver a jugar sin reiniciar el programa

El código está organizado de la siguiente manera:

Diccionario de palabras: almacena categorías y palabras usando tuplas

Función elegir_palabra(): selecciona aleatoriamente una categoría y palabra

Función pedir_letra(): valida la entrada del usuario

Función procesar_acierto(): verifica si la letra está en la palabra

Función jugar(): controla la lógica principal del juego

Bloque principal (__main__): permite repetir el juego

Tecnologías utilizadas
Lenguaje de programación: Python
Librería estándar: random
Paradigma: Programación estructurada
Entorno: Consola / Terminal
