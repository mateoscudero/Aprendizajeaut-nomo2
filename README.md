# Aprendizajeaut-nomo2
Trabajo autonomo 2

Juego del Ahorcado

Descripción:

Juego del ahorcado en Python donde el jugador debe adivinar una palabra letra por letra antes de quedarse sin intentos. Cada letra correcta se revela y cada error resta un intento.
